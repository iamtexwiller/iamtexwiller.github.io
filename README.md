This project is an interactive interface that blends **technical visualization** with **developer jokes**.  
Inspired by the first Rust CVE in the Linux kernel, but reimagined as a showcase of humor, certifications, and social links.


## 🚀 Features

- **Interactive cards** with technical info and jokes  
- **Modal pop-ups** for detailed (and funny) explanations  
- **Vertical certification list** with icons and official links/PDFs  
- **Social media section** with humorous captions  
- Modern CSS layout with gradients, badges, and colorful pills  


## 📂 Structure

- `index.html` → Main page structure  
- `style.css` → Visual styles (cards, pills, modals)  
- `script.js` → JS functions for modal interactivity  
- `redhat-certificado.pdf` → Red Hat Delivery Specialist certificate
